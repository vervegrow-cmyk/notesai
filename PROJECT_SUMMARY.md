## 🚀 快速开始（3 步）

### 1️⃣ 启动本地开发

```bash
cd cc_test
npm install
npm run dev
```

访问 `http://localhost:5173` 测试应用

### 2️⃣ 推送到 GitHub

```bash
git add .
git commit -m "feat: Complete AI valuation tool"
git push origin main
```

### 3️⃣ 部署到 Vercel

1. 访问 [Vercel Dashboard](https://vercel.com/dashboard)
2. 导入 GitHub 仓库
3. 添加环境变量 `KIMI_API_KEY`
4. 点击部署

**部署完成！** 🎉
