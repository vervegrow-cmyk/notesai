### 3. 后端 API

#### 认证相关
```
POST /api/auth/login       # 登录
POST /api/auth/register    # 注册
POST /api/auth/logout      # 登出
POST /api/auth/verify      # 验证Token
```

#### 询价管理
```
POST /api/inquiry/list           # 获取询价列表（支持筛选）
POST /api/inquiry/get            # 获取单个询价详情
POST /api/inquiry/save           # 保存/创建询价
POST /api/inquiry/update-status  # 更新询价状态
POST /api/inquiry/update         # 编辑询价信息
POST /api/inquiry/delete         # 删除询价
POST /api/inquiry/statistics     # 获取统计数据
```

## 💾 数据存储

当前系统使用**内存存储**（JavaScript Map）：
- ✅ 开发和测试速度快
- ✅ 无需数据库配置
- ❌ 应用重启后数据丢失
- ❌ 无法多进程运行

**生产环境建议集成数据库**（MongoDB、PostgreSQL等）

## 🐛 常见问题

### Q: 为什么看不到询价数据？
A: 系统默认使用内存存储。关闭应用后数据会丢失。请确保：
   1. 已登录
   2. 后端API正常运行（3001端口）
   3. 检查浏览器控制台是否有错误

### Q: 忘记密码怎么办？
A: 当前系统只有硬编码的管理员账户。生产环境需要实现密码重置功能。

### Q: 如何修改默认账户？
A: 编辑 `features/auth/service.js`，修改初始用户设置：
```javascript
users.set('admin', { username: 'admin', password: '新密码', role: 'admin' });
```

### Q: 为什么无法启动应用？
A: 检查：
   1. Node.js已安装（v14+）
   2. npm依赖已安装（npm install）
   3. 端口3001和5173未被占用
